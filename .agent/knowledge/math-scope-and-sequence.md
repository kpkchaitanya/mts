# MTS Math Scope and Sequence — by Grade

**Organization:** Masters Tuition Services LLC
**Version:** v1
**Status:** Active
**Source:** NC Common Core Math Standards (aligned)

---

## Purpose

This document defines the expected math topics by grade level for MTS students.
Agents generating math content consult this document to ensure:
* Topics are grade-appropriate
* Prerequisites are respected
* Progression follows the curriculum arc

---

## Grade 2

| Domain | Topics |
|--------|--------|
| Operations & Algebraic Thinking | Addition/subtraction to 20; arrays; basic multiplication concepts |
| Number & Operations in Base Ten | Place value to 1000; adding/subtracting 3-digit numbers |
| Measurement | Length (cm, in); telling time; counting money |
| Geometry | Basic shapes; partitioning shapes into halves/thirds/fourths |

---

## Usage Notes for Content Generation

* Always confirm grade from the declared grade attribute — never infer from content difficulty.
* Word problems should use culturally neutral, age-appropriate contexts.
* For Grades 9–10, SAT-style problem formats are encouraged (multiple-choice + grid-in).
* Difficulty within a worksheet should progress: warm-up → core practice → challenge.
