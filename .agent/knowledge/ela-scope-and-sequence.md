**Organization:** Masters Tuition Services LLC
**Version:** v1

**Source:** NC Common Core ELA Standards (aligned)

## Purpose

This document defines the expected ELA topics by grade level for MTS students.
Agents generating ELA content consult this document to ensure:
* Passages and questions are grade-appropriate
* Skills targeted match the grade's curriculum arc
* Reading complexity (Lexile) is in the right range

## Grade 2–3: Foundational Reading & Writing

| Strand | Skills |
|--------|--------|
| Reading Literature | Story elements (character, setting, problem); main idea; retelling |
| Reading Informational | Key details; author's purpose; text features (headings, captions) |
| Writing | Narrative; opinion with reason; organizing ideas |
| Language | Nouns/verbs/adjectives; capitalization; punctuation; spelling patterns |
| Vocabulary | Context clues; word families; compound words |

| Strand | Skills |
|--------|--------|

| Writing | Informative; opinion/argumentative; research basics |

| Vocabulary | Greek/Latin roots; multiple meanings; connotation |

| Strand | Skills |
|--------|--------|
| Reading Literature | Complex theme; irony; narrative technique; intertextual connections |
| Reading Informational | Rhetoric; evaluating evidence; synthesizing multiple sources |

| Language | Sentence errors (comma splices, fragments, run-ons); transitions; precision |
| SAT Reading | Command of evidence; words in context; paired passages; information graphics |
| SAT Writing | Expression of ideas; standard English conventions |

## Question Type Progression

For any grade, ELA questions should target:

| Level | Question Type | Example |
|-------|---------------|---------|
| Literal | Direct recall | "What does the author say about...?" |
| Inferential | Reading between the lines | "What can we infer about...?" |
| Evaluative | Judgment / synthesis | "Why does the author choose...?" |
| Vocabulary | Word meaning in context | "As used in line X, 'urgent' most nearly means..." |

Distribute question types across all three levels in every passage set.
Grades 2–3: heavily literal + inferential.
Grades 8–10: balanced across all four types.

---

## Usage Notes for Content Generation

* Passages must not contain disturbing, adult, or politically charged content.
* Informational passages should use real-world topics appropriate for the grade.
* Fiction passages should be culturally diverse and inclusive.
* Always verify that the Lexile estimate for a passage matches the grade range above.
